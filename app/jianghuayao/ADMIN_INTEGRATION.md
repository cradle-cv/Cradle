## 📋 集成步骤（仅需5分钟）

### Step 1：创建目录

```powershell
cd D:\cradle

# 创建 admin 子目录
mkdir app/jianghuayao/admin/heritage
mkdir app/jianghuayao/admin/imagery
```

### Step 2：复制管理组件

**文件 1 & 2：复制到 `components/` 目录**

- `heritage-management.js` → `app/jianghuayao/components/heritage-management.js`
- `imagery-management.js` → `app/jianghuayao/components/imagery-management.js`

**复制方式**：
1. 打开文件
2. Ctrl+A 全选 → Ctrl+C 复制
3. VSCode 新建对应文件 → Ctrl+V 粘贴 → Ctrl+S 保存

### Step 3：复制管理页面

**文件 3 & 4：复制到各自的目录**

- `admin/heritage/page.js` → `app/jianghuayao/admin/heritage/page.js`
- `admin/imagery/page.js` → `app/jianghuayao/admin/imagery/page.js`

### Step 4：刷新浏览器

按 `Ctrl+F5` 强制刷新，清除缓存

### Step 5：测试功能

1. 打开浏览器进入后台：`http://localhost:3000/jianghuayao/admin`
2. 点击"非遗项目管理" → "进入管理"
3. 点击"新增项目" → 填写信息 → "添加"
4. 返回首页查看效果

## 🔧 数据存储

### 存储方式
使用浏览器 **localStorage**：
- **非遗项目**：`heritageProjects`
- **江华映象**：`imageryItems`

### 特点
✅ **无需后端** - 前端存储，立即生效
✅ **快速响应** - 秒级反应
✅ **不占用空间** - 数据存在本地

⚠️ **局限**：
- 仅在当前浏览器生效
- 清除缓存会丢失数据
- 多人协作不方便

### 升级建议
未来可升级为：
- MongoDB/MySQL 数据库
- 用户认证系统
- 数据备份功能

## 💡 常见问题

**Q: 添加的数据保存在哪里？**
A: 保存在浏览器的 localStorage 中，这台电脑这个浏览器访问时能看到。

**Q: 能在手机上编辑吗？**
A: 可以，但手机上的数据和电脑上是分离的。建议在电脑上统一管理。

**Q: 数据会丢失吗？**
A: 不会丢失，除非清除浏览器缓存。建议定期备份。

**Q: 可以分享给其他人吗？**
A: 未来升级为数据库后可以。目前每个电脑是独立的。
